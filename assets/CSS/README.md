# Comandos Básicos de Git

## git --version

Muestra la versión instalada de Git.

Ejemplo:

```bash id="jjlwm5"
git --version
```

---

## git init

Inicializa un repositorio Git en la carpeta actual.

Ejemplo:

```bash id="6jlwm5"
git init
```

---

## git add

Agrega archivos al área de preparación.

Ejemplo:

```bash id="jlwm5y"
git add .
```

---

## git commit

Guarda cambios con un mensaje descriptivo.

Ejemplo:

```bash id="jlwm5z"
git commit -m "Primer commit"
```

---

## git status

Muestra el estado actual del repositorio.

Ejemplo:

```bash id="0zjlwm"
git status
```
